# 📝 Editable To-Do List Web App

An interactive and editable To-Do List web application built using **HTML, CSS, and JavaScript**.
This project allows users to manage daily tasks efficiently with a clean and modern UI.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed (custom green checkbox)
* 💾 LocalStorage support (tasks remain after refresh)
* ⌨️ Add task using Enter key
* 🎨 Modern glassmorphism UI design
* 📱 Responsive layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Custom Checkbox Styling)
* JavaScript (DOM Manipulation & LocalStorage)
